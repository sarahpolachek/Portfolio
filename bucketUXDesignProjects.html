UX Projects

Leaderboard Design

This design challenge was created within a three hour time limit. Its leaderboard layout provides instant information on everything from age range, weight bracket, and competitors' rank.

Users are able to quickly narrow search results to see information on matches won.

Timer Activity Challenge

The concept of this app was to create a quick timer that users could use to time their workouts.

Users have the ability to choose a workout, choose an amount of time for working out, view how much time has passed, and how many calories were burned.

Intranet - Microsoft Sharepoint

This project cannot be shared publicly, however, I helped build Victorian Finance's internal employee website.

This project was based in Microsoft Sharepoint and was handed off to the company with skeleton wireframes. My role in this project consisted of helping fill in and build out each webpage using wireframes and css customization.

The goal was to create a similar look and feel across the site by using branded colors and fonts and making sure the placement of content and elements reflected similarly across each page.

Cafe Cravings

This website was built for a Pittsburgh Coffee Shop, located in Chatham Village of Mount Washington.

The coffee shop offers vintage, calming vibes and this website was built to reflect the welcoming atmosphere that the shop offers. This website also relfects the menu brochure I developed for them, which can be viewed on the Visual Designs page of this site.

The goal of this project was to provide a simple website that would house necessary information such as a menu and contact information. Both owners wanted to keep the project small and simple.
